# 2.2 GDPR 사전 동의 설정

### GDPR 개요

* **GDPR(General Data Protection Regulation)**&#xC740; European Union(EU)의 개인정보 보호 법령입니다.
* 유럽 경제 지역(EEA) 사용자에 대해서는 개인정보 및 광고 식별자 수집·활용에 대해 **사전 동의**를 받아야 합니다.

### 적용 대상

{% hint style="info" %}
* 앱 사용자가 **유럽 경제 지역(EEA)**&#xC5D0; 속하는 경우
* EEA 외 지역 사용자에게는 동의 팝업이 표시되지 않음
{% endhint %}

### 동의 수집 방식

{% hint style="info" %}
* EEA 사용자 동의는 Google AdMob의 User Messaging Platform(UMP SDK)를 통해 처리됩니다.
* UMP SDK 를 사용하지 않는 광고 네트워크를 운영하는 경우 별도의 인터페이스(setGdpr)를 제공합니다.
  * 예: Cauly, Levelplay 등
{% endhint %}

### 필수 설정 사항(중요)

{% hint style="warning" %}
**1) 스토어 정책 설정과 SDK 설정 일치**

* 플랫폼 콘솔에 설정한 데이터 수집 항목과 SDK에서 실제 동작하는 개인정보/광고 설정이 반드시 일치해야 합니다.
  * Android → Google Play Console의 데이터 보안 및 광고설정
  * iOS → App Store Connect
* 설정 불일치 시:
  * 정책 위반 가능
  * 광고 제한(Limited Ads) 발생 가능

**2) Initialize를 진행 후 호출해야 합니다.**
{% endhint %}

```java
AdWhaleMediationAds.init(this, object : AdWhaleMediationOnInitCompleteListener {
    override fun onInitComplete(statusCode: Int, message: String) {
        Log.i(MainActivity::class.java.simpleName, ".onInitComplete($statusCode, $message);")
    }
})

AdWhaleMediationAds.requestGdprConsent(this, object : AdWhaleGdprConsentListener {
    override fun onResult(isSuccess: Boolean, message: String) {
        val result = "GDPR Consent: ${if (isSuccess) "Success" else "Failed"}, $message"
        Log.d(MainActivity::class.java.simpleName, result)
    }
})
```

# 1. 시작하기 전에

본 문서는 ADwhale Mediation SDK를 파트너사의 Flutter 안드로이드 어플리케이션에 연동하기 위한 가이드 문서입니다. 연동전에 다음의 안내 사항을 반드시 읽어 주시기 바랍니다.

{% hint style="info" %}
본 문서는 최신 ADwhale Mediation SDK 버전을 기준으로 업데이트 되고 있습니다.

하위 버전의 기술지원은 별도 문의 바랍니다.
{% endhint %}

{% tabs %}
{% tab title="Android" %}
### **Android 요구사항**

* 안드로이드 지원 버전
  * Android API Level 24 이상
  * Android 5.0(API 21) 미만의 경우 multi-dex 적용 필요
* 안드로이드 타겟 버전 지원
  * API Level 34
  * GooglePlay의 타겟 API 수준 요구사항 충족하기
* Support Library
  * AndroidX 대응
  * Android Support Library

{% hint style="danger" %}
**Google Play 정책 안내**

**2025년 8월 31일부터** Google Play에 제출되는 모든 앱은 **Android 15 (API Level 35) 이상**을 `targetSdkVersion`으로 지정해야 합니다.

* 참고: [https://developer.android.com/google/play/requirements/target-sdk?hl=ko](https://developer.android.com/google/play/requirements/target-sdk?hl=ko)

***

**2025년 11월 11일부터** Android 15 이상을 타겟팅하는 모든 앱은 **64비트 기기에서 16KB페이지 크기를 지원**해야 합니다.

* 참고: [https://developer.android.com/guide/practices/page-sizes?hl=ko](https://developer.android.com/guide/practices/page-sizes?hl=ko)
{% endhint %}
{% endtab %}

{% tab title="iOS" %}
### **iOS 요구사항**

{% hint style="danger" %}
**Flutter** 프로젝트는 **Cocoapod 기반 Swift**가 기본 네이티브 언어로 사용됩니다.

사용하는 프로젝트 구조에 맞게 가이드를 적용해 주세요.
{% endhint %}

* 최소 지원 사양
  * iOS 13.0 이상
  * Xcode 15.0 이상
  * CocoaPods

{% hint style="info" %}
2025년 4월 24일부터 앱 스토어에 앱을 제출하려면 **Xcode 16.0 이상 버전 사용이 필요합니다.**&#x20;

[**https://developer.apple.com/news/upcoming-requirements/?id=02212025a**](https://developer.apple.com/news/upcoming-requirements/?id=02212025a)
{% endhint %}

{% hint style="danger" %}
**Xcode15** 이상에서 빌드할 경우, **iOS Deployment Target Version(최소 지원 버전)을 13.0 이상으로 설정**합니다.

iOS 13.0 이상으로 설정하지 않을 경우, 특정 광고 네트워크의 초기화 과정에서 비정상 종료(Crash)가 발생할 수 있습니다.
{% endhint %}

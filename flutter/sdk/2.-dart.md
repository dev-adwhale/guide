### 최신 SDK version : 2.7.4+0

### Version Usage Guide

프로젝트 상황에 따라 아래 방식을 권장합니다.

#### Android SDK 버전에 고정하여 사용하는 경우 (권장)

운영 안정성을 중시하는 경우

```
{
  adwhale_sdk_flutter: ~2.7.4+0
}
```

* Flutter 핫픽스만 자동 반영
* `>=2.7.4+0 <2.8.0` 범위

#### 특정 버전을 완전히 고정하여 사용하는 경우

```
{
  adwhale_sdk_flutter: 2.7.4+0
}
```

* 버전 변경 없음
* 재현성, 장애 분석에 최적
* QA / 장애 재현 환경
* 장기 유지 브랜치

#### Troubleshooting

Android 빌드 오류 발생 시:

```
cd android
./gradlew clean
```

#### 환경점검

```
flutter doctor
```

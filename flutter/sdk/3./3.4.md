### MainActivity 클래스

{% hint style="danger" %}
**중요!** MainActivity 클래스를 FlutterActivity가 아닌 FlutterFragmentActivity로 사용해야 합니다.&#x20;

FlutterActivity 를 사용할 경우 광고 노출에 제한이 있을 수 있습니다.
{% endhint %}

```java
public class MainActivity extends FlutterFragmentActivity {
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
    }
}
```

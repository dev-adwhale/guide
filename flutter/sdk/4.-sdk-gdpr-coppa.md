{% tab title="초기화" %}
### 초기화

{% hint style="info" %}
광고를 요청하기 전에 초기화가 이루어져야 합니다. 초기화 방법은 초기화를 담당하는 **AdWhaleMediationAds 클래스를 main() 에서 호출**해주며, 초기화 **호출 결과는 return 값을 통해 통지**됩니다. 자세한 구현은 아래 API와 샘플코드를 참고하여 구현하면 됩니다.
{% endhint %}

**AdWhaleMediationAds 클래스**

```dart
Future<AdWhaleInitResult> initialize() async
```

**초기화 샘플코드**

```dart
import 'package:adwhale_sdk_flutter/adwhale_sdk_flutter.dart';

// 앱 시작 시 SDK 초기화
void main() {
  WidgetsFlutterBinding.ensureInitialized();
  final result = await AdWhaleMediationAds.instance.initialize();
  debugPrint('AdWhale SDK 초기화 결과: $result');
  if (result.isSuccess) {
    runApp(const MyApp());
  }
}
```

{% tab title="로거설정" %}
### **로거 설정**

{% hint style="info" %}
로거 설정 기능은 현재 **Android**에서만 지원됩니다. iOS에서 호출 시 `Unsupported` 에러가 반환됩니다. iOS 지원은 추후 예정입니다. 참고 부탁드립니다.
{% endhint %}

**AdWhaleMediationAds 클래스**

```dart
Future<void> setLoggerEnabled(bool enabled) async
```

| 파라미터 타입 | 파라미터 값   |
| ------- | -------- |
| bool    | 로그 레벨 설정 |

**로그 레벨 설정**

| 값     | 설명                      |
| ----- | ----------------------- |
| false | 모든 로그 출력 안함             |
| true  | Error, Warn, Info 로그 출력 |

```dart
Future<String> getLogLevel() async // 현재 설정된 로그레벨 출력
```

**로거설정 샘플코드**

```dart
import 'package:adwhale_sdk_flutter/adwhale_sdk_flutter.dart';

void main() {
  // setLoggerEnabled 호출
  await AdWhaleMediationAds.instance.setLoggerEnabled(true);

  // 테스트: setLoggerEnabled 호출 후 getLogLevel 확인
  final logLevel = await AdWhaleMediationAds.instance.getLogLevel();
  print('현재 로그 레벨: $logLevel');
}
```

{% tab title="GDPR/COPPA 설정" %}
### GDPR, COPPA설정

{% hint style="info" %}
GDPR은 유럽연합(이하 'EU')의 개인정보 보호 법령으로 서비스 제공자는 EU 사용자의 개인정보 또는 그에 준하는 정보에 대한 수집 및 활용에 대해 사용자에게 동의 여부를 확인받아야 합니다. GDPR 규정 준수 및 규정에 따른 광고네트워크 동작에 필요한 절차입니다. 유럽 경제 지역 (European Economic Area) 사용자에게 동의 여부를 선택할 수 있는 기능을 제공하며 애드몹 UMP (User Messaging Platform)에 의해서 처리됩니다.
{% endhint %}

**AdWhaleMediationAds 클래스**

```dart
Future<Map<String, Object?>> requestGdprConsent() async //GDPR 동의 선택화면 호출
```

```dart
Future<void> resetGdprConsentStatus() async //GDPR 설정 리셋
```

```dart
Future<void> setCoppa(bool isChildDirected) async // coppa (아동 보호) 설정
```

**GDPR, Coppa 샘플코드**

```dart
// request gdpr
final result = await AdWhaleMediationAds.requestGdprConsent();
print('GDPR Request: ${result.isSuccess ? 'Success' : 'Failed'}\n${result.message}');

// reset consent
await AdWhaleMediationAds.instance.resetGdprConsentStatus();

// set coppa
await AdWhaleMediationAds.instance.setCoppa(true);
```

# 4.1 로거설정

### **로거 설정**

{% hint style="info" %}
로거 설정 기능은 현재 **Android**에서만 지원됩니다. iOS에서 호출 시 `Unsupported` 에러가 반환됩니다. iOS 지원은 추후 예정입니다. 참고 부탁드립니다.
{% endhint %}

**AdWhaleMediationAds 클래스**

```dart
Future<void> setLoggerEnabled(bool enabled) async
```

| 파라미터 타입 | 파라미터 값   |
| ------- | -------- |
| bool    | 로그 레벨 설정 |

**로그 레벨 설정**

| 값     | 설명                      |
| ----- | ----------------------- |
| false | 모든 로그 출력 안함             |
| true  | Error, Warn, Info 로그 출력 |

```dart
Future<String> getLogLevel() async // 현재 설정된 로그레벨 출력
```

**로거설정 샘플코드**

```dart
import 'package:adwhale_sdk_flutter/adwhale_sdk_flutter.dart';

void main() {
  // setLoggerEnabled 호출
  await AdWhaleMediationAds.instance.setLoggerEnabled(true);

  // 테스트: setLoggerEnabled 호출 후 getLogLevel 확인
  final logLevel = await AdWhaleMediationAds.instance.getLogLevel();
  print('현재 로그 레벨: $logLevel');
}
```

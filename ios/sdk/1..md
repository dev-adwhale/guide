# 1. 요구사항 및 사용권한

<mark style="color:$info;">본 문서는 AdWhale SDK를 파트너사의 IOS 어플리케이션에 연동하기 위한 가이드 문서입니다. 연동전에 반드시 다음의 안내 사항을 반드시 읽어주시기 바랍니다.</mark>

{% hint style="info" %}
본 문서는 최신 AdWhale SDK 버전을 기준으로 업데이트 되고 있습니다.

하위 버전의 기술지원은 별도 문의 바랍니다.
{% endhint %}



### 요구사항 <a href="#undefined" id="undefined"></a>

* 최소 지원 사양
  * iOS 13.0 이상
  * Xcode 16.0 이상
  * CocoaPods

{% hint style="info" %}

  [**https://developer.apple.com/news/upcoming-requirements/?id=02212025a**](https://developer.apple.com/news/upcoming-requirements/?id=02212025a)

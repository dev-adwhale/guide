# 1. 요구사항 및 사용권한

본 문서는 AdWhale SDK를 React-Native로 개발된 파트너사의 iOS 어플리케이션에 연동하기 위한 가이드 문서입니다. 연동전에 다음의 안내 사항을 반드시 읽어 주시기 바랍니다.

{% hint style="info" %}
본 문서는 최신 AdWhale SDK 버전을 기준으로 업데이트 되고 있습니다.

하위 버전의 기술지원은 별도 문의 바랍니다.
{% endhint %}

### 요구사항

{% hint style="danger" %}
**React Native CLI** 프로젝트는 **Objective-C**, **Expo** 프로젝트는 **Swift**가 기본 네이티브 언어로 사용됩니다.

사용하는 프로젝트 구조에 맞게 가이드를 적용해 주세요.
{% endhint %}

* 최소 지원 사양
  * iOS 12.0 이상
  * Xcode 15.0 이상
  * CocoaPods

{% hint style="info" %}
2025년 4월 24일부터 앱 스토어에 앱을 제출하려면 **Xcode 16.0 이상 버전 사용이 필요합니다.**&#x20;

[**https://developer.apple.com/news/upcoming-requirements/?id=02212025a**](https://developer.apple.com/news/upcoming-requirements/?id=02212025a)
{% endhint %}

{% hint style="danger" %}
**Xcode15** 이상에서 빌드할 경우, **iOS Deployment Target Version(최소 지원 버전)을 12.0 이상으로 설정**합니다.

iOS 12.0 이상으로 설정하지 않을 경우, 특정 광고 네트워크의 초기화 과정에서 비정상 종료(Crash)가 발생할 수 있습니다.
{% endhint %}

### Expo 환경

{% hint style="warning" %}
Expo 환경에서 구현하는 경우 ios 폴더가 없을 수 있습니다.

이에따라 Podfile을 포함한 ios 설정 파일이 표시되지 않을 수 있습니다.

이 경우, [expo 환경 설정](/broken/pages/N59i5aEP52lbDkMxcH08) 과정을 진행하여 ios 폴더를 생성한 후 작업을 진행해주시기 바랍니다.
{% endhint %}

# 4.2 GDPR 사전 동의 설정

### GDPR 개요

* **GDPR(General Data Protection Regulation)**&#xC740; European Union(EU)의 개인정보 보호 법령입니다.
* 유럽 경제 지역(EEA) 사용자에 대해서는 개인정보 및 광고 식별자 수집·활용에 대해 **사전 동의**를 받아야 합니다.

### 적용 대상

{% hint style="info" %}
* 앱 사용자가 **유럽 경제 지역(EEA)**&#xC5D0; 속하는 경우
* EEA 외 지역 사용자에게는 동의 팝업이 표시되지 않음
{% endhint %}

### 동의 수집 방식

{% hint style="info" %}
* EEA 사용자 동의는 Google AdMob의 User Messaging Platform(UMP SDK)를 통해 처리됩니다.
* UMP SDK 를 사용하지 않는 광고 네트워크를 운영하는 경우 별도의 인터페이스(setGdpr)를 제공합니다.
  * 예: Cauly, Levelplay 등
{% endhint %}

### 필수 설정 사항(중요)

{% hint style="warning" %}
**1) 스토어 정책 설정과 SDK 설정 일치**

* 플랫폼 콘솔에 설정한 데이터 수집 항목과 SDK에서 실제 동작하는 개인정보/광고 설정이 반드시 일치해야 합니다.
  * Android → Google Play Console의 데이터 보안 및 광고설정
  * iOS → App Store Connect
* 설정 불일치 시:
  * 정책 위반 가능
  * 광고 제한(Limited Ads) 발생 가능

**2) Initialize를 진행 후 호출해야 합니다.**
{% endhint %}

**AdWhaleMediationAds 클래스**

```typescript
requestGdprConsent(): Promise<{ isSuccess: boolean; message: string }> //GDPR 동의 선택화면 호출
```

```typescript
resetGdprConsentStatus() //GDPR 설정 리셋
```

{% hint style="info" %}
* setGdpr은 UMP SDK 를 사용하지 않는 광고 네트워크를 위해 호출
* setGdpr 기능은 현재 **Android**에서만 지원됩니다. iOS에서 호출 시 `Unsupported` 에러가 반환됩니다. iOS 지원은 추후 예정입니다. 참고 부탁드립니다.
{% endhint %}

```typescript
setGdpr(consent: boolean)
```

**샘플코드**

```dart
// request gdpr
const result = await AdWhaleMediationAds.requestGdprConsent();
console.log(`GDPR Request: ${result.isSuccess ? 'Success' : 'Failed'}\n${result.message}`);

// reset consent
AdWhaleMediationAds.resetGdprConsentStatus();

// set gdpr 
AdWhaleMediationAds.setGdpr(true);
```
